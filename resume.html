Project Deliverable 2

Assignment 1: Yahtzee

Overview

Yahtzee is a classic dice game that involves rolling five dice to achieve specific combinations and earn points. The game consists of 13 rounds, and in each round, the player can roll the dice up to three times to achieve the desired combination.

Architecture

The Yahtzee game is implemented as a web application using HTML, CSS, and JavaScript.

Game flow

Game Start: The game initializes with a score card, dice and a roll button on a hand made custom background. Rolling Dice: - Clicking the roll button rolls the Dice - The dice animation is displayed and the dice numbers are displayed Scoring: - The player selects a combination of the dice to score based on the dice values. - Once the score combnation is selected the scorecard is updated End of round: - Once the score combination is selected, the total score is updated and the player progresses to the next round. Game End: - After the 13 rounds are played the game is done and the final score is displayed.

Conclusion

The yahtzee game is a fun web based game that allows a player to pass some time and learn to play a common dice game!

Assignment 2: Yahtzee Node.JS Client/Server implementation

Overview

Yahtzee is a classic dice game that involves rolling five dice to achieve specific combinations and earn points. The game consists of 13 rounds, and in each round, the player can roll the dice up to three times to achieve the desired combination.

Architecture

The Yahtzee game is implemented as a web application using HTML, CSS, and JavaScript for the frontend. The backend is powered by Node.js and Express, allowing communication between the client and server.

Frontend

Front end styling, the game is rendered and allows for user interaction. The front end is done with HTML and CSS that allows for the structure and styling of the game interface Front end game logic, the logic is implemented with JavaScript, allowing for the dice rolling and communication with the server

Back end

The backend manages the game state, handles dice rolling logic, and communicates with the frontend. Key components include: - Node.js: Provides the runtime environment for the server. - Express: A web application framework for Node.js that simplifies server-side development.
